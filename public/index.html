Air BnC Portfolio endpoint catalog

GET /api/properties

Returns an array of properties objects


                    {
                        "properties": [
                          {
                            "property_id": <id>,
                            "property_name": <name>,
                            "location": <location>,
                            "price_per_night": <price>,
                            "host": <host name>,
                            "image": <img_url>
                          },
                          {
                            "property_id": <id>,
                            "property_name": <name>,
                            "location": <location>,
                            "price_per_night": <price>,
                            "host": <host name>,
                            "image": <img_url>
                          }
                    }

Optional queries

  • ?maxprice=<max cost per night>
  • ?minprice=<min cost per night>
  • ?sort=<cost_per_night | popularity>
  • ?order=<ascending | descending>
  • ?host=<id>
  • ?amenity=<amenity_slug>

Try me : /api/properties

Try me : /api/properties?maxprice=100

GET /api/properties/:id/reviews

Returns an array of reviews objects


                    {
                        "reviews": [
                          {
                            "review_id": <id>,
                            "comment": <name>,
                            "rating": <location>,
                            "created_at": <date>,
                            "guest": <guest name>,
                            "guest_avatar" <img_url>
                          },
                          ...
                        ],
                        "average_rating": <avg of ratings>
                      }

Try me : /api/properties/:id/reviews

GET /api/properties/:id

Returns a single property object


                    {
                        "property":  {
                            "property_id": <id>,
                            "property_name": <name>,
                            "location": <location>,
                            "price_per_night": <price>,
                            "description": <description>,
                            "host": <host name>,
                            "host_avatar" <image_url>,
                            "favourite_count": <count>,
                            "image": <img_url>,
                            "amenities": <amenities array>
                      }
                    }

Optional query: ?user_id=<id> returns if id has favourited the property or not


                    {
                        "property":   {
                         // regular fields...,
                         favourited: <true | false>
                        }
                      }

Try me : /api/properties/1

Try me : /api/properties/1?user_id=1

POST /api/properties/:id/reviews

Inserts a new row into the reviews table

Expects the following payload as input : 


                    {
                        "guest_id": <id>,
                        "rating": <rating>,
                        "comment": <comment>
                    }

Responds with the following : 


                    {
                        "review_id": <id>,
                        "property_id": <id>,
                        "guest_id": <id>,
                        "rating": <rating>,
                        "comment": <comment>,
                        "created_at": <date>
                    }
