Air BnC Portfolio endpoint catalog

GET /api/properties

Returns an array of properties 


                    {
                        "properties": [
                          {
                            "property_id": <id>,
                            "property_name": <name>,
                            "location": <location>,
                            "price_per_night": <price>,
                            "host": <host name>,
                            "image": <img_url>
                          },
                          {
                            "property_id": <id>,
                            "property_name": <name>,
                            "location": <location>,
                            "price_per_night": <price>,
                            "host": <host name>,
                            "image": <img_url>
                          }
                    }

Optional queries

  • ?maxprice=<max cost per night>
  • ?minprice=<min cost per night>
  • ?sort=<cost_per_night | popularity>
  • ?order=<ascending | descending>
  • ?host=<id>
  • ?amenity=<amenity_slug>

Try me : /api/properties

Try me : /api/properties?maxprice=100

GET /api/properties/:id

Returns a single property 


                    {
                        "property":  {
                            "property_id": <id>,
                            "property_name": <name>,
                            "location": <location>,
                            "price_per_night": <price>,
                            "description": <description>,
                            "host": <host name>,
                            "host_avatar" <image_url>,
                            "favourite_count": <count>,
                            "image": <img_url>,
                            "amenities": <amenities array>
                      }
                    }

Optional query: ?user_id=<id>

Returns if id has favourited the property or not


                    {
                        "property":   {
                         // regular fields...,
                         favourited: <true | false>
                        }
                      }

Try me : /api/properties/1

Try me : /api/properties/1?user_id=1

DELETE /api/properties/:id/users/:id/favourite

Removes the appropriate favourite data from the database

Returns 'no body'

POST /api/properties/:id/favourite

Adds a favourite property to the database

Exprects the following payload as input:


                    {
                        "guest_id": <id>
                    }

Responds with the following


                    {
                        "msg": "Property favourited successfully.",
                        "favorite_id": <id>
                    }
